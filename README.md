# WB Dashboard (Streamlit)

Дашборды и аналитика для работы с маркетплейсом Wildberries.

## Приложение «Реклама маркетплейса» (RK)

Аналитика рекламы по дням: загрузка Excel-отчёта WB (раздел «По дням»), KPI, целевой CPL, окупаемость (ROMI, ДРР, маржа), рекомендации.

### Запуск

**Через скрипт (macOS):**
```bash
./launch_rk.command
```

**Вручную:**
```bash
cd RK
pip install streamlit pandas openpyxl
streamlit run app_rk.py --server.port 8509
```

Откроется браузер: http://localhost:8509

### Данные

В боковой панели загрузите Excel-отчёт WB с разделом «По дням». После загрузки доступны вкладки: Аналитика по дням, Калькулятор метрик, Планировщик продаж.

## Остальные модули

В репозитории могут присутствовать другие приложения и скрипты (телеграм-бот, отчёты и т.д.). Зависимости см. в `requirements.txt` или в скриптах запуска.
